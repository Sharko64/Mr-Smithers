# Usage

## Installation

> [!Important]
> fix to use cURL piped into shell

## Arch Linux

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers
bash init.sh --arch
```

## Debian Linux

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers
bash init.sh --debian
```

## macOS

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers
bash init.sh --macOS
```

## Windows (WSL)

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers
bash init.sh --windows-WSL
```
