# Mr. Smithers
## Configuration setup tool for workstation/server state preservation.

These scripts are meant to preserve configurations states of provided operating systems.

## Arch Linux

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers
bash init.sh --arch
```

## Debian Linux

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers
bash init.sh --debian
```

## macOS

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers
bash init.sh --macOS
```

## Windows

```bash
gh repo clone Sharko64/Mr-Smithers
cd Mr-Smithers

```
