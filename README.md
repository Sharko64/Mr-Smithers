# Configuration Setup Tool

These scripts are meant to preserve configurations states of provided operating systems.

## Arch Linux

```bash
gh repo clone Sharko64/Configuration-Setup
cd Configuration-Setup
bash init.sh --arch
```

## Debian Linux

```bash
gh repo clone Sharko64/Configuration-Setup
cd Configuration-Setup
bash init.sh --debian
```

## macOS

```bash
gh repo clone Sharko64/Configuration-Setup
cd Configuration-Setup
bash init.sh --macOS
```

## Windows

```bash
gh repo clone Sharko64/Configuration-Setup
cd Configuration-Setup
bash init.sh --windows
```
