# Configuration Setup Tool

These scripts are meant to preserve configurations states of provided operating systems.

## Arch Linux

```bash
gh clone <repo>
cd <repo>
bash init.sh --arch
```

## Debian Linux

```bash
gh clone <repo>
cd <repo>
bash init.sh --debian
```

## Windows

```bash
gh clone <repo>
cd <repo>
bash init.sh --windows
```

## macOS

```bash
gh clone <repo>
cd <repo>
bash init.sh --macOS
```
